ピクセルグリッドについて

ピクセルグリッドはJavaScriptの会社です。ECMAScriptという国際標準に基づいた言語であるJavaScriptを中心にhtml5やFlashといった技術を積極的に利用したWebアプリケーションを提供していきます。

JavaScript/Ajaxに関するご相談やiPhone/iPad/Androidで動作するWebサービス、Webサイトの制作も承ります。

業務案内

株式会社ピクセルグリッドはインターネットを進化させ、より便利に機能的にしていきたいと考えています。
具体的には大きく2種類に分けられます。新技術利用・研究のための自社Webサービス開発や運営と、高度な技術を使ったWebサイト制作です。

Webサービス開発・運営

主にHTML5関連技術、JavaScript/Ajaxを多用した次世代インターフェイスの実験的サービスを開発・運営しています。

フロントエンド・エンジニア

HTML5、CSS、JavaScriptを利用したスマートフォン向けおよびPC向けWebサイト、Webアプリの制作を中心に行います。

具体的にどういった技術を採用して目的を達成するかという提案や、実装方法の相談から実際の制作まで広く関わることができます。

必須スキル・経験

  • HTML、CSSの仕様書に目を通し、HTML、CSSの思想を理解している
  • JavaScriptの言語仕様を理解し、DOMを操作するスクリプトを記述できる

優遇スキル・経験

  • jQueryを使ったWebサイト、Webアプリケーションの実装経験がある
  • Web制作会社での実務経験
  • JavaScript以外の言語の実装経験、知識があること
  • デザインに関する知識があること
  • ユーザーエクスペリエンスを考慮した画面設計の知識があること

node.js エンジニア

node.jsを利用したサーバサイドの実装を行い、Webサービスを構築します。

自社サービスの実装から、受託制作での新しい表現の実験、提案などnode.jsを使ったコンテンツを幅広く作成していただきます。

必須スキル・経験

  • node.jsの基本的な特徴や利点を理解し、実際になにか作ったことがある
  • HTML, CSSの思想を理解している
  • JSONの仕様と利用方法を理解している
  • MVCアーキテクチャの理解がある
  • 他の言語でサーバサイドのアプリを作成したことがある
  • フレームワークを利用した開発経験があるかた

優遇スキル・経験

  • NoSQLを利用したことがあること
  • web socketを利用したデモやアプリケーションを作ったことがある人
  • npmにモジュールをアップしたことがある人
  • サーバ・インフラ運用（クラウド含む）の経験がある人
